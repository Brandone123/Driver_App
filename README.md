# My Android Expo App

## Installation

1. Clone the project from the Git repository:

   ```shell
   git clone <REPOSITORY_URL>
   ```

2. Install the required dependencies:

   ```shell
   npm install
   ```

3. Run the app:

   ```shell
   npm start
   ```

## Prerequisites

- Node.js
- Expo CLI
- Android Emulator
